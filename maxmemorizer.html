          Help & Instructions
          Overview

          This application allows you to randomly select URLs from a predefined list with customizable settings.

          Features
          • URL Prefix/Suffix: Add prefixes or suffixes to all URLs before they open.
          • Default Frequency: Set the default frequency for new items.
          • Adjust Probabilities: Control the likelihood of selecting from Bucket1 or Bucket2.
          • Draw History: Log of the last 5 drawn URLs.
          • Undo Last Draw: Revert the last draw action.
          • Import/Export State: Save/load your current state using JSON.
          • Dark Mode: Toggle a dark theme.
          • Fun Cards & Session: Up to 5 “Fun Cards” for Bucket3 sessions.
          • Bucket 3 Session Progress: See how many session cards remain at any time.
          How to Use
          1. Set URL Prefix/Suffix and click "Save URL Prefix/Suffix".
          2. Default Frequency: "Edit" → enter a positive integer → "Save".
          3. Probabilities: Use the slider for Bucket1 vs Bucket2.
          4. Generate Session: Fill Bucket3 with a random blend of study/fun items.
          5. Draw from Bucket3: Removes next card from session, opening it in a new tab.
          6. Default Session Cards: How many draws to generate for each session.
          7. Define Fun Cards: 1–5 custom fun URLS. Blank fields are ignored.
          8. Draw & Undo: Draw from Bucket1/Bucket2, or revert with "Undo Last Draw".
          9. Import/Export/Reset: Manage your state with these controls.
          10. Dark Mode: Toggle the dark theme at any time.
          11. I’m Feeling Lucky®: Randomly picks between Bucket3 or Bucket1/Bucket2, and also changes its own button color distinctly from the page background.
